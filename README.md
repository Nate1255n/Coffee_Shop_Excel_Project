# Coffee Sales Dashboard:

## Coffee Sales Analysis — Excel Data Analytics Project
This project analyzes transactional coffee sales data spanning from 2019 to 2022 across three countries — the United States, Ireland, and the United Kingdom. 
The dataset contains 1,000 customer records, 48 products across four coffee types (Arabica, Excelsa, Liberica, and Robusta), and a multi-year orders history 
with details on roast type, bag size, unit price, and loyalty card membership.
